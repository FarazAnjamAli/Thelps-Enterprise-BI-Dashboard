# Thelps-Enterprise-BI-Dashboard
# Thelps Ecommerce Data Warehouse & BI System

An enterprise-level Data Warehouse and Business Intelligence project developed using Power BI, Tableau, SQL, and ETL processes for advanced ecommerce analytics and decision-making.

---

# Project Overview

This project transforms raw ecommerce operational data into a centralized analytical Data Warehouse capable of supporting business intelligence reporting and executive decision-making.

The system includes:

- Data Warehouse Modeling
- ETL Processing
- SQL Analytics
- Power BI Dashboards
- Tableau Dashboards
- KPI Reporting
- Forecasting & Trend Analysis
